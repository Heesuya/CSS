선택자

선택자
HTML 문서 내부에서 스타일을 적용하고싶은 요소(HTML태그)를 선택할 때 사용하는 기술
선택자로 요소를 선택해서 원하는 스타일을 적용

전체선택자

HTML문서 안의 모든 태그를 선택할 때 사용하는 선택자

 * {속성 : 값;}

아이디 선택자

HTML문서 내에 해당하는 속성 id를 가진 태그만 선택할 때 사용(id 중복되면 안됨)

#아이디{속성 : 값;}
  • 아이디 선택자 테스트1
  • 아이디 선택자 테스트2
  • 아이디 선택자 테스트3
  • 아이디 선택자 테스트4
  • 아이디 선택자 테스트5

클래스 선택자

HTMP 문서 내에 여러개 태그를 동일 클래스이름으로 지정하여 선택할 때 사용

.클래스{속성 : 값;}
  • 클래스 선택자 테스트1
  • 클래스 선택자 테스트2
  • 클래스 선택자 테스트3
  • 클래스 선택자 테스트4
  • 클래스 선택자 테스트5

태그선택자

HTML 문서 안의 동일한 태그를 모두 선택할 때 사용하는 선택자

태그이름{속성 : 값;}
