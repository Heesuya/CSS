테두리 스타일 지정

border-style : 테두리 스타일 지정

선택자{border-[방향]-style : none | hidden | dashed | dotted | double | groove | inset | outset | ridge | solid}

border-width : 테두리 두께 지정

선택자{border-[방향]-width : thin | medium | thick | px}

border-color : 테두리 색상 지정

선택자{border-[방향]-color : 색상값}

border : 테두리 스타일 한번에 지정

선택자{border-[방향] : width style color}

border-radius : 박스의 모서리를 둥글게 하는 속성

선택자{border-[방향]-radius : 숫자}

border-shadow : box 그림자 주는 속성

여백스타일

margin : 요소와 요소 사이의 거리

        선택자{margin-[위치] : 거리;} -> top,right,bottom,left를 선택해서 거리를 지정
        선택자{margin : 거리;} -> 4방향 한번에 설정
        선택자{margin : 거리1 거리2;} -> 거리1 : top,bottom 설정 / 거리2 : left right 한번에 설정
        선택자{margin : 거리1 거리2 거리3 거리4;} -> 거리1 : top / 거리2 : right / 거리3 : bottom / 거리4 : left

padding : 테두리와 컨체츠 사이의 거리

        선택자{padding : 거리;} -> 4방향 한번에 설정 / 다른옵션은 마진과 동일
