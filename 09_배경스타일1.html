배경스타일

background-color : 요소의 배경색을 지정하는 속성

선택자{background-color : 색상이름 | #16rgb | rgb(r,g,b)}

background-clip : 배경색의 범위를 지정하는 속성

선택자{background-clip : border-box | padding-box | content-box}

background-image : 요소의 배경을 이미지로 설정

선택자{background-image : url(이미지경로)}

실제 전달하고싶은 메시지

background-repeat : 배경이미지 반복처리 속성

선택자{background-repeat : repeat | repeat-x | repeat-y | no-repeat}

background-position : 배경이미지의 위치를 조정

선택자{background-position : x축위치 y축위치}

background-size : 배경이미지 크기 조정

선택자{background-size : auto | contain | cover | 크기지정}

background-origin : 배경이미지 배치 기준

선택자{background-origin : border-box | padding-box | content-box}

background-attachment : 페이지 상하 이동 시 배경이미지 고정여부

선택자{background-attachment : scroll | fixed}
